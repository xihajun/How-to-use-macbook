## Link
We have 3 ways to do this in markdown language.
1. use [your text](link) [my homepage](www.edu-ing.cn)
2. use [your text][num or text] with [number or text]: yourlink in the end of your context. 
* eg. [myhomepage][1]
* eg. [github][helloworld]
[1]
